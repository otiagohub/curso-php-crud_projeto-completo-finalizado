# Projeto Fly By Night completo (front-end e back-end)

**Obs.:** você precisa ter um ambiente local de desenvolvimento que suporte PHP e MySQL.

Recomendo usar o **XAMPP** ou **WampServer**.

- Baixe o repositório
- Descompacte para pasta C:\xampp\htdocs (ou a pasta do seu servidor web local)
- Renomeie a pasta para **curso-php-crud_completo**
- Com o servidor web ligado (Apache no XAMPP), acesse no navegador: localhost/curso-php-crud_completo

Atenção: não se esqueça de criar o banco de dados via localhost/phpmyadmin e importar para ele as tabelas e dados usando o arquivo `flybynight_estoque.sql` contido na pasta `dumps`.
